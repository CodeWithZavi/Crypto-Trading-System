# Crypto Trading System (MERN Stack)

A professional crypto trading simulation platform.

## Features
- **Authentication**: Secure Login/Signup with MongoDB.
- **Dashboard**: Real-time TradingView Chart, Mock Market Data.
- **Trading**: Buy and Sell Crypto with logical validation and portfolio updates.
- **Wallet**: View demo balance, asset holdings, and transaction history.
- **Tech Stack**: React, TailwindCSS, Node.js, Express, MongoDB.

## Prerequisites
- Node.js installed.
- MongoDB installed and running locally on port 27017.

## How to Run

1.  **Start MongoDB** (ensure `mongod` is running).

2.  **Start Backend**:
    ```bash
    cd server
    npm install # if not already installed
    npm run dev
    ```
    Server runs on `http://localhost:5000`.

3.  **Start Frontend**:
    ```bash
    cd client
    npm install # if not already installed
    npm run dev
    ```
    Client runs on `http://localhost:5173`.

4.  **Open Browser**: Go to `http://localhost:5173`.
